# Stratégie réseaux sociaux : choisir, publier, mesurer sans t'éparpiller

Une stratégie réseaux sociaux n'est pas un planning de publication. C'est un cadre qui relie cible, message, format, plateforme et objectif.

## Les questions de départ

- où ton audience est-elle vraiment attentive ;
- quel type de contenu sais-tu tenir dans la durée ;
- cherches-tu visibilité, preuve, conversation ou vente ;
- quelle suite proposes-tu derrière.

## La structure la plus simple

### Une plateforme principale

Là où tu concentres l'essentiel de l'effort.

### Deux ou trois piliers de contenu

Exemples : pédagogie, coulisses, preuve.

### Un format de conversion

Newsletter, prise de rendez-vous, page ressource, offre.

## Les KPI les plus utiles

- clics sortants ;
- réponses ou messages ;
- conversions assistées ;
- contenus qui amènent des conversations qualifiées.

Une bonne stratégie sociale n'essaie pas d'impressionner tout le monde. Elle essaie d'être lisible pour les bonnes personnes.
