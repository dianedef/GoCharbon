Enrichir une liste email ne consiste pas à empiler des colonnes “au cas où”. Le but est d’ajouter juste assez d’informations pour mieux segmenter et mieux agir.

## Ce qu’il vaut la peine d’ajouter

- site web ;
- entreprise ;
- localisation ;
- téléphone si pertinent ;
- source ;
- date d’enrichissement.

## À quoi ça sert

- mieux personnaliser ;
- mieux segmenter ;
- éliminer les doublons ;
- choisir le bon canal de contact ;
- prioriser les leads.

## Bonne méthode

1. pars d’une base propre ;
2. enrichis par lots ;
3. vérifie un échantillon ;
4. conserve la source ;
5. segmente ensuite.

## Ce qu’il faut éviter

- garder toutes les données retournées ;
- mélanger données certaines et données fragiles ;
- prospecter sans contrôle qualité.

Une petite base bien enrichie vaut mieux qu’un gros fichier bourré d’erreurs.
