Ajouter des numéros de téléphone à une liste n’a d’intérêt que si le résultat reste exploitable. Sinon, tu obtiens juste plus de bruit.

## Quand ça vaut le coup

- prospection locale ;
- qualification rapide ;
- relance commerciale ;
- segmentation plus fine.

## La bonne logique

1. pars d’une base claire ;
2. enrichis avec une source cohérente ;
3. vérifie la qualité sur un échantillon ;
4. distingue les numéros sûrs des numéros incertains ;
5. adapte le canal selon le contexte.

## Ce qu’il faut garder

- numéro ;
- type de numéro si possible ;
- source ;
- date ;
- niveau de confiance.

## Ce qu’il faut éviter

- appeler des numéros non qualifiés à la chaîne ;
- mélanger mobile, standard et téléphone de support ;
- croire qu’un volume énorme compense une mauvaise base.

En enrichissement, la qualité bat presque toujours la quantité.
