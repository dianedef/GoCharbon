Un site WordPress n’est jamais “inviolable”. En revanche, tu peux le rendre beaucoup moins facile à casser, ce qui élimine déjà une énorme partie du risque.

## Les bases qui comptent vraiment

- mises à jour régulières ;
- plugins limités ;
- mots de passe solides ;
- 2FA ;
- sauvegardes ;
- hébergement propre ;
- accès admin réduits.

## Là où ça casse souvent

- vieux plugins ;
- thèmes abandonnés ;
- comptes admin oubliés ;
- environnement mal maintenu.

## Ce qu’il faut faire d’abord

1. supprime l’inutile ;
2. mets à jour ;
3. sauvegarde ;
4. limite les accès ;
5. surveille les logs et alertes.

La sécurité WordPress n’est pas un plugin miracle. C’est surtout de l’hygiène sérieuse et régulière.
