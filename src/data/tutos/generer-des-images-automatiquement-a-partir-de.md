Quand un site publie régulièrement, refaire les visuels à la main devient vite absurde. L’idée n’est pas d’automatiser pour faire plus de bruit. L’idée est d’automatiser ce qui se répète.

## Ce qu’il faut récupérer

- titre ;
- extrait ;
- image de base ;
- catégorie ;
- URL ;
- éventuellement auteur ou date.

## Workflow simple

1. récupère les données depuis WordPress ;
2. envoie-les vers une source intermédiaire ;
3. mappe-les dans un template visuel ;
4. exporte selon le canal.

## Ce qu’il faut standardiser

- les dimensions ;
- la hiérarchie du texte ;
- les couleurs ;
- les variantes de format.

Une bonne automatisation visuelle garde la cohérence. Une mauvaise produit juste des images génériques en série.
