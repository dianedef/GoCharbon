- **Rapport client**
    
    Tenez vos clients informés à tout moment grâce à des rapports clients professionnels entièrement modifiables. Le rapport client est un moyen idéal pour impressionner vos clients en allant au-delà de leurs attentes et en leur faisant comprendre pourquoi vous leur facturez la maintenance.
    
    ## **Construisez la confiance avec vos clients**
    
    
    
    ### **Option marque blanche**
    
    Les rapports clients peuvent être exportés au format .PDF et totalement en marque blanche. Votre marque, votre rapport pour vos clients.
    
    ### **Certification des données**
    
    Nous pouvons certifier vos données de mise à jour, de performance et de maintenance de votre client. Cette certification par un tiers renforcera vos relations et votre confiance avec vos clients existants.
    
    ### **Disponible en plusieurs langues**
    
    Le rapport client est disponible en anglais, français. Envoyez-nous un message si votre langue est manquante !
    
- Comment changer la langue du rapport de maintenance ?
    
    Les rapports de maintenance peuvent être édités en anglais ou en français.
    
    Par défaut, les rapports de maintenance sont édités en anglais. Vous avez deux façons de changer cela.
    
    # **Changer la langue du rapport de maintenance**
    
    
    
    Étape 1 : cliquez sur le projet de votre choix dans le tableau de bord de WP Umbrella.
    
    
    
    **Étape 2** : Cliquez sur l'onglet Rapports.
    
    **Étape 3** : Cliquez sur Paramètres
    
    **Étape 4** : Sélectionnez une nouvelle langue et cliquez sur enregistrer, en bas de la page.
    
- Comment créer un rapport client en marque blanche
    
    Vous ne voulez peut-être pas toujours que vos clients sachent ce que vous utilisez pour gérer leurs sites Web, il n'y a aucun mal à cela. C'est pourquoi nous avons créé une intégration en marque blanche pour votre rapport client.
    
    Je vais vous montrer comment en créer un :
    
    **Étape 1 : accédez à [l'application WP Umbrella](https://app.wp-umbrella.com/) et sélectionnez le projet pour lequel vous souhaitez créer un rapport client en cliquant sur son nom.**
    
    
    
    **Étape 2 : Cliquez sur « Signaler » puis sur « Paramètres » et assurez-vous que l'option marque blanche est cochée.**
    
    
    
    **Étape 3 : mettez à jour vos paramètres**
    
    Et voilà, votre rapport client sera libre de toute mention à WP Umbrella.
    
- Comment créer un rapport de maintenance
    
    Renforcez votre relation avec vos clients en leur envoyant des rapports de maintenance professionnels sur ce que vous avez fait.
    
    Le rapport client est un moyen idéal pour établir la confiance avec vos clients et leur montrer le bon travail de maintenance que vous avez effectué.
    
    # **Les données disponibles dans le rapport de maintenance de WP Umbrella**
    
    Il y a des données que nous ajoutons automatiquement dans le rapport de maintenance telles que :
    
    - Taux de disponibilité.
    - Performances moyennes.
    - Aperçu de la vitesse de la page Google.
    - Nombre de thèmes et plugins mis à jour (en un coup d'œil et avec les détails).
    - Les contrôles de sécurité (noyau WordPress à jour, certificat SSL, indexabilité par les moteurs de recherche)
    - Le calendrier (ex : 01 août 2021 - 31 août 2021) .
    
    Et il y a les données que vous pouvez personnaliser telles que :
    
    - Les informations de votre entreprise - elles apparaîtront sur tous vos rapports.
    - La langue du rapport (l'anglais et le français sont disponibles pour le moment mais envoyez-nous un message si vous souhaitez que nous ajoutions votre langue).
    - Un espace texte pour laisser un message personnalisé à votre client.
    - La possibilité de marquer le rapport en marque blanche.
    - La possibilité de certifier vos données avec WP Umbrella
    
    **Pourquoi certifier les données avec WP Umbrella ?**
    
    Faire certifier vos données par une application tierce peut améliorer la confiance entre votre client et vous.
    
    # **Comment créer un rapport client**
    
    **Etape 1** : allez dans [l'application WP Umbrella](https://app.wp-umbrella.com/) et sélectionnez le projet pour lequel vous souhaitez créer un rapport client en cliquant sur son nom.
    
    
    
    
    
    **Étape 2 :**  Cliquez sur " Signaler " puis sur " Paramètres " et ajoutez les détails et le logo de votre entreprise. Vous pouvez également choisir un code de couleur personnalisé pour les titres afin de refléter votre image de marque.
    
    **Étape 3** : Ajoutez un message de bienvenue pour que vos clients se sentent spéciaux. Cela sera automatiquement ajouté à tous vos rapports.
    
    Décidez ensuite si vous souhaitez que le rapport soit en marque blanche et/ou si vous souhaitez que les données soient certifiées par WP Umbrella.
    
    
    
    **Étape 3** : Cliquez sur les **paramètres de mise** à **jour,** puis sur **générer un rapport de maintenance** .
    
    **Étape 4:**  Vous pouvez ajouter votre travail personnalisé ou toute information personnalisée que vous souhaitez. Sélectionnez ensuite une période et cliquez sur Créer le rapport.
    
    
    
    Vous serez automatiquement redirigé vers la page de présentation des rapports où vous pourrez accéder à votre rapport au format .PDF.
    
- Comment masquer le plugin de WP Umbrella
    
    Vous ne voulez peut-être pas toujours que vos clients sachent ce que vous utilisez pour gérer leurs sites Web, il n'y a aucun mal à cela. C'est pourquoi nous avons créé une fonctionnalité White Label pour masquer notre plugin.
    
    Voici comment cela fonctionne :
    
    **Étape 1 : Connectez-vous à votre administrateur WordPress**
    
    **Étape 2 : accédez à Apparence > Éditeur de thème.**
    
    
    
    **Étape 3 :  Sélectionnez le fichier functions.php**
    
    
    
    **Étape 4 : ajoutez cette ligne de code**
    
    ```
    add_filter('wp_umbrella_is_white_label', '__return_true');
    
    ```
    
    
    
    **Étape 5 : Cliquez sur « Mettre à jour le fichier »**
    
    Et c'est tout! Le plugin de WP Umbrella est maintenant caché !
