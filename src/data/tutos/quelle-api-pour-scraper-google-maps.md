- Saviez-vous que [Scrap.io](http://Scrap.io) dispose d'une API ?
- Elle vous permet d'accéder aux fonctionnalités de l'app sans passer par l'interface.
- Voici 3 raisons de s'y intéresser:
- **1.** La première : Vous pouvez scripter la recherche de leads. Par exemple, si vous avez beaucoup de recherches à passer, vous pouvez créer un petit programme qui va s'en charger pour vous, sans avoir besoin de passer par l'interface et créer les exports 1 par 1.
- Mieux : si le format des exports [Scrap.io](http://Scrap.io) ne vous convient pas, vous pouvez _créer votre propre format d'export_. Ou programmer un _transfert automatique_ vers votre CRM.
- **2.** La plupart des outils de cold-emailing disposent d'une API. Vous pouvez donc automatiser complètement votre prospection, de A à Z.
- Vous pouvez créer un script qui va _rechercher des leads sur [Scrap.io](http://Scrap.io)_, les _envoyer à votre outil de prospection_ et _lancer une campagne en automatique_.
- Ensuite, vous avez juste à traiter e-mails de réponse qui arrivent dans votre boîte de réception, **pendant que vos robots prospectent pour vous !**
- **3.** Enfin, vous pouvez **proposer les données de [Scrap.io](http://Scrap.io)** (et Google Maps) de manière _totalement transparente_ (et invisible) à vos propres clients.
- C'est une manière simple de rajouter une _killer-feature_ à une app, ou de proposer un **super service** à vos clients à moindre effort.
