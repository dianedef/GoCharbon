Raccourcir un lien ne sert pas seulement à le rendre plus joli. Le vrai intérêt, c’est de garder une logique propre sur :

- les UTM ;
- le tracking ;
- les variantes de campagne ;
- les liens réutilisables.

## Ce que Switchy peut simplifier

- centraliser les liens ;
- poser les bons paramètres ;
- suivre les clics ;
- garder des liens plus lisibles.

## Bonne pratique

Définis une convention simple pour :

- la source ;
- le support ;
- la campagne ;
- le nom du lien.

Le vrai gain vient de la cohérence. Pas juste du raccourcissement.
