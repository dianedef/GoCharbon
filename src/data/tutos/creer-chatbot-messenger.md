## Comment fonctionne un chatbot Facebook Messenger ? 

Le fonctionnement d’un **chatbot Messenger** peut différer selon le type du bot que vous créez. Sachez qu’il existe deux principales catégories de robots de chat. 

### Les chatbots simples

Ils sont programmés pour puiser dans une bibliothèque de réponses pré-élaborées. Ils envoient donc le message au client en suivant le schéma classique du WorkFlow. Les interactions sont limitées avec ce type d’outils, mais ils permettent de **bien guider les conversations**. 

### Les chatbots intelligents polyvalents utilisant l’IA

Ce sont des bots qui sont communément appelés les « robots évolués » compte tenu du **NLP**, de l’intelligence artificielle et du machine learning qu’ils utilisent.

Le NLP est une technologie de traitement du langage naturel qui permet à ces assistants virtuels de générer des messages clairs et précis, se rapprochant d’une discussion avec un humain.

Grâce à ces robots, vous donnerez à vos acheteurs l’impression de discuter directement avec un agent du service client. Ces chatbots intelligents sont plus polyvalents.
