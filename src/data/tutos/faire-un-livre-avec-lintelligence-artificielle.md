L’IA peut accélérer l’écriture d’un livre. Elle ne remplace ni ta pensée, ni ton angle, ni ton expérience.

Le bon usage est simple : **elle t’aide à produire, pas à tricher sur le fond**.

## Ce qu’elle peut très bien faire

- t’aider à cadrer le sujet ;
- proposer un plan ;
- reformuler ;
- résumer ;
- générer des variantes de titres ;
- t’aider en réécriture.

## Ce qu’elle fait mal

- inventer une vraie expertise ;
- raconter une expérience vécue crédible ;
- garantir la justesse des faits ;
- produire une voix vraiment singulière.

## Workflow utile

1. définis le lecteur et la promesse du livre ;
2. construis un plan solide ;
3. rédige chapitre par chapitre ;
4. injecte tes exemples, tes cas, tes positions ;
5. réécris lourdement ;
6. fais une vraie relecture éditoriale.

## Le bon usage des prompts

Demande à l’IA :

- d’élargir un plan ;
- d’ordonner des idées ;
- de proposer des transitions ;
- de reformuler un passage trop lourd.

Évite de lui demander “écris-moi le livre entier”.

## Le vrai risque

Le problème n’est pas seulement le style IA. Le problème, c’est le vide. Un livre qui paraphrase ce qu’on trouve déjà partout n’a pas d’intérêt, même s’il est bien tourné.

Utilise donc l’IA pour accélérer la fabrication, pas pour esquiver la pensée.
