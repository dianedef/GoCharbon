L’accessibilité web ne se résume pas à cocher une case. Mais tu peux déjà améliorer énormément les choses sans lancer un chantier monstre.

## Commence par les fondamentaux

- contraste correct ;
- navigation clavier ;
- textes lisibles ;
- structure de titres claire ;
- labels de formulaires ;
- alternatives textuelles utiles.

## Ce qu’il faut éviter

- penser uniquement “outil d’audit” ;
- cacher les problèmes derrière un widget magique ;
- écrire du contenu compliqué puis espérer qu’un plugin règle le sujet.

L’accessibilité utile commence par un site plus clair, plus stable et plus navigable.
