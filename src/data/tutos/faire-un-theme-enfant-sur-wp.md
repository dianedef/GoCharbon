Si tu touches au code d’un thème WordPress directement, tu prépares juste le moment où une mise à jour effacera ton travail.

## À quoi sert un thème enfant

Il te permet de :

- surcharger du CSS ;
- ajouter un peu de PHP ;
- modifier certains templates ;
- garder tes personnalisations après mise à jour du thème parent.

## Quand tu en as besoin

- tu touches au code ;
- tu modifies un template ;
- tu veux garder un historique propre.

## Quand tu n’en as pas besoin

- si tu fais tout via le customizer ;
- si tu ne touches jamais au thème ;
- si tu utilises seulement des options natives ou un page builder.

Le thème enfant n’est pas compliqué. Il évite surtout de bricoler au mauvais endroit.
