Rafraîchir une page automatiquement peut être pratique pour une surveillance, un dashboard ou un suivi temps réel. Ça peut aussi être très pénible si c’est mal utilisé.

## Quand ça se justifie

- suivi de métriques ;
- monitoring ;
- tableau de bord ;
- page d’état.

## Ce qu’il faut éviter

- rafraîchir une page de lecture normale ;
- faire perdre la position de lecture ;
- surcharger inutilement le navigateur ou le serveur.

L’automatisation est bonne quand elle sert l’usage réel. Pas quand elle donne juste l’impression de “temps réel”.
