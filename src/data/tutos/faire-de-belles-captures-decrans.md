Une mauvaise capture d’écran peut ruiner une explication simple. Le but n’est pas de faire du design pour le plaisir. Le but est de rendre l’information claire en un coup d’œil.

## Ce qu’une bonne capture doit montrer

- le bon cadrage ;
- le bon contraste ;
- l’élément important bien visible ;
- aucune distraction inutile.

## Les bons réflexes

- ferme les onglets ou notifications inutiles ;
- zoome si besoin ;
- ajoute une annotation légère ;
- garde une cohérence visuelle d’une capture à l’autre.

## Ce qu’il faut éviter

- un écran surchargé ;
- des captures floues ;
- des encadrés partout ;
- un style différent à chaque image.

Une belle capture d’écran est surtout une capture compréhensible.
