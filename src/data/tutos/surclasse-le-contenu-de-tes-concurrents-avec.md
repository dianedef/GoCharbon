Tu ne dépasses pas un concurrent en ajoutant 500 mots médiocres. Tu le dépasses en répondant mieux.

## Ce qu’il faut regarder

- intention de recherche ;
- angle ;
- profondeur ;
- clarté ;
- preuves ;
- exemples ;
- fraîcheur.

## Bonne méthode

1. lis les pages déjà visibles ;
2. note ce qu’elles oublient ;
3. construis un plan plus net ;
4. ajoute du concret ;
5. améliore ensuite le maillage et le titre.

Le levier principal n’est pas “plus de texte”. C’est “meilleure réponse”.
