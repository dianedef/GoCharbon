L’automatisation Facebook peut aider à gagner du temps. Elle ne doit pas transformer ton acquisition en machine à messages douteux.

## Ce qui peut être automatisé proprement

- réponses simples ;
- qualification légère ;
- relances ;
- diffusion contrôlée de contenu.

## Ce qu’il faut garder humain

- les échanges sensibles ;
- la proposition commerciale ;
- la personnalisation utile ;
- les cas ambigus.

Automatiser le répétitif est utile. Automatiser la relation entière est rarement une bonne idée.
