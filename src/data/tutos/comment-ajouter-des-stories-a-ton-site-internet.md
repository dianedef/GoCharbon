# Comment Ajouter Des Stories A Ton Site Internet

## WEB STORIES : LE FORMAT QUI CAPTE L'ATTENTION

### tl;dr

Les stories ne sont plus reservees a Instagram. Tu peux ajouter ce format immersif sur ton site web pour augmenter l'engagement et meme ton SEO. Google adore les Web Stories et les met en avant dans Discover. Ce tuto te montre les methodes, du plugin WordPress aux solutions pour sites custom.

---

## Pourquoi mettre des stories sur ton site ?

- **Engagement** -- Le format swipe vertical est addictif. Les visiteurs restent plus longtemps.
- **SEO** -- Les Web Stories Google sont indexables et apparaissent dans Discover.
- **Mobile-first** -- 70% de ton trafic est mobile. Les stories sont faites pour les pouces.
- **Differenciation** -- Tres peu de sites utilisent ce format.

## Methode 1 : Google Web Stories sur WordPress

Le plugin officiel "Web Stories" de Google est gratuit et toujours activement maintenu (version 1.42.0 en mai 2025, 90 000+ installations actives).

1. Installe le plugin "Web Stories" par Google depuis le repertoire WordPress (gratuit, open-source sous licence Apache 2.0)
2. Va dans Stories > Ajouter
3. Utilise l'editeur drag-and-drop : ajoute des pages (chaque page = un ecran swipable), insere images, videos, textes et animations
4. Ajoute un CTA sur la derniere page
5. Publie -- la story a sa propre URL indexable
6. Integre sur ta page d'accueil avec le bloc "Web Stories"
7. Necessite WordPress 6.6 ou plus recent

**SEO :** remplis titre, description et image de couverture. Chaque page doit avoir du texte. Le plugin gere le balisage schema automatiquement. Les Web Stories peuvent apparaitre dans Google Discover et dans les resultats de recherche Google en format carrousel.

## Methode 2 : Storyly (widget style Instagram)

Storyly ajoute des stories en bulles en haut de ta page (web et app), exactement comme Instagram.

1. Inscris-toi sur [storyly.io](https://storyly.io)
2. Cree ton groupe de stories avec slides : images, videos, boutons, sondages, quiz, comptes a rebours, codes promo
3. Copie le code JavaScript et integre-le dans ton site (ou SDK pour les apps mobiles)

**Prix (mis a jour 2025) :** Storyly est desormais une plateforme enterprise. Plan "Core Engagement Suite" a partir de 600 $/mois (1 placement, stories et banners, stickers interactifs basiques). "Personalization & Widget Suite" a partir de 1 000 $/mois (3 placements, tous les widgets, contenu shoppable). "Comprehensive Orchestration Suite" sur devis (placements illimites, IA, integrations). C'est un outil pour les e-commercants et apps a gros volume -- pas pour les petits blogs. Utilise par des marques comme Domino's, Decathlon et d'autres entreprises de grande taille.

## Methode 3 : StorifyMe (alternative)

Meme principe que Storyly avec un editeur visuel et des elements interactifs (quiz, sondages, compte a rebours). Tarification sur devis -- contacter [storifyme.com](https://www.storifyme.com). Plus accessible que Storyly pour les PME.

## Methode 4 : Coder avec Zuck.js (gratuit)

Pour les sites custom, utilise la librairie open-source Zuck.js qui reproduit l'interface stories d'Instagram :

1. Cree tes visuels sur Canva (1080x1920)
2. Integre Zuck.js dans ton site
3. Configure le swipe et la progression automatique
4. Personnalise couleurs, timing et callbacks

## Quel contenu mettre dans tes stories ?

- **E-commerce :** nouveaux produits, promos flash, avis clients, tutos produit
- **Blog :** resume d'articles en 5-6 slides, chiffres cles, coulisses
- **SaaS :** nouvelles fonctionnalites, tutos rapides, temoignages
- **Services :** avant/apres, presentation equipe, FAQ visuelles

## Bonnes pratiques

- **5 a 10 pages par story** -- Plus court ca manque, plus long les gens decrochent
- **Une idee par page** -- 1-2 phrases max, texte minimum 24px
- **Les 2 premieres pages comptent le plus** -- Capte l'attention immediatement
- **Video > Image > Texte seul** -- Le mouvement retient l'attention
- **Termine par un CTA clair**

## Astuces GoCharbon

- **Les Web Stories Google sont un hack SEO sous-exploite.** Peu de sites en font, Google les pousse dans Discover.
- **Recycle tes stories Instagram.** Adapte-les pour ton site en 5 minutes.
- **Met les stories au-dessus de la ligne de flottaison.** Si le visiteur doit scroller, personne ne les verra.
- **Analyse les taux de completion.** Si les gens quittent a la page 3 sur 8, le contenu decroche.
- **Change tes stories regulierement.** Des stories datees de 3 mois donnent l'impression d'un site abandonne.
