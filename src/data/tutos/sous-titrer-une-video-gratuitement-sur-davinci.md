Les sous-titres ne servent pas seulement à faire joli sur les réseaux. Ils améliorent la compréhension, retiennent l’attention et rendent la vidéo plus accessible.

## Workflow simple

1. importe la vidéo ;
2. génère ou ajoute les sous-titres ;
3. relis ;
4. corrige le timing ;
5. exporte selon ton canal.

## Ce qu’il faut vérifier

- lisibilité ;
- taille ;
- contraste ;
- fautes ;
- rythme.

Des sous-titres moyens valent souvent mieux que rien. Des sous-titres faux ou illisibles, non.
