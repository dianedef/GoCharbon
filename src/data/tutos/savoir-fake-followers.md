Tu n’as pas besoin d’un rapport de 40 pages pour soupçonner de faux followers. Quelques signaux suffisent souvent.

## Ce qu’il faut regarder

- ratio abonnés / engagement ;
- qualité des commentaires ;
- évolution brutale du nombre d’abonnés ;
- cohérence de l’audience avec la niche ;
- répétition de profils suspects.

## Ce qu’il faut éviter

- juger un compte sur un seul post ;
- croire qu’un gros chiffre vaut autorité ;
- collaborer sans vérifier un minimum.

Le but n’est pas d’être obsédé par la fraude. Le but est d’éviter de payer pour une audience vide.
