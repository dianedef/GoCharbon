Personne ne peut te promettre qu’un compte Meta ne sera jamais restreint. En revanche, tu peux éviter une bonne partie des erreurs qui déclenchent des contrôles ou des blocages.

## Commence par les bases

- utilise ton vrai nom et un profil réel ;
- active l’authentification à deux facteurs ;
- vérifie ton business manager ;
- garde des coordonnées de facturation cohérentes ;
- évite de créer dix comptes d’un coup “au cas où”.

## Respecte les politiques publicitaires

Avant de lancer une campagne, relis les règles sur :

- les promesses trop fortes ;
- les secteurs sensibles ;
- les avant/après ;
- les attributs personnels ;
- les pages de destination trompeuses.

Source officielle : [Meta Advertising Standards](https://www.facebook.com/policies/ads/).

## Les signaux qui posent souvent problème

- compte neuf qui scale trop vite ;
- moyen de paiement douteux ou instable ;
- page peu crédible ou vide ;
- domaine récent sans vraie présence ;
- créas agressives ou borderline.

## Bonne pratique de lancement

Quand un compte est récent :

1. commence avec des campagnes simples ;
2. limite les changements brusques de budget ;
3. évite les structures trop complexes dès le jour 1 ;
4. vérifie manuellement chaque créa avant publication.

## Si tu es restreint

- lis précisément le motif ;
- corrige le point bloquant ;
- prépare des justificatifs propres ;
- demande une revue seulement quand le dossier est propre.

Le bon réflexe n’est pas de “recréer un autre compte”. C’est souvent le meilleur moyen d’aggraver le problème.
