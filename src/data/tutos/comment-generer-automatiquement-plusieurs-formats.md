En moyenne, les entreprises publient généralement une fois par jour sur Instagram, ce qui signifie que vous devez créer beaucoup d'images pour rester pertinent.

Voici quelques statistiques qui devraient définitivement vous convaincre de poster plus régulièrement sur Instagram :

- 90 % des utilisateurs suivent au moins un compte professionnel.
- 130 millions d'utilisateurs cliquent sur le bouton d'achat chaque mois.

Pour rappel, les formats Instagram les plus courants sont :

- Carré (1:1) : 1080px x 1080px
- Paysage (1,91:1) : 1080px x 608px
- Portrait (4:5) : 1080px x 1350px
- Histoire (9:16) : 1080px x 1920px

Imaginons que vous souhaitiez poster chaque jour une histoire, un carré et un paysage. Cela représente plus de 20 images à créer chaque semaine. Mais comment créer des images sans y passer des heures ?

Avec [**Abyssale**](https://fr.abyssale.com/) ces tâches fastidieuses et répétitives deviennent un jeu d'enfant ! Dans cet article, je vais couvrir toutes les étapes nécessaires pour automatiser cette production 😎.

# **Créer une marque dynamique template sur Abyssale**

Vous pouvez soit créer un template de toutes pièces, soit en choisir un dans notre [**bibliothèquetemplate**](https://fr.abyssale.com/creative-automation/templates/) .

Dans ce guide, nous allons utiliser un template de notre bibliothèque :**"Promo - Produit unique"**

# **1. Sélectionnez un template**

Connectez-vous à votre [**compteAbyssale**](https://app.abyssale.com/) et accédez à la page de la **bibliothèqueTemplates** (vous pouvez accéder à la bibliothèque template à partir de la navigation supérieure), puis sélectionnez le produit template **"Promo - Produit unique**" dans la bibliothèque.


Utilisez un template de la bibliothèque publique

# **2. Personnalisez le template**

Dans le cadre de ce tutoriel, vous pouvez laisser tout en l'état ou mettre à jour les mises en page et les composants par défaut, tels que le logo, l'image d'arrière-plan et le texte, en y ajoutant la marque de votre entreprise.

Ne vous inquiétez pas, ils pourront toujours être modifiés lorsque nous générerons les bannières. Faire ces changements maintenant vous évitera de télécharger votre logo plusieurs fois par exemple.

# **3. Ajout de nouveaux formats à la base template**

Pour ajouter de nouvelles formas à un template , il suffit de cliquer sur le bouton **Ajouter un nouveau format** dans la section des formats du bannière (panneau latéral gauche).

Vous serez invité à choisir entre un format prédéfini ou à définir vos propres dimensions personnalisées.


Abyssale template éditeur - Ajout de formats

Une fois que vous aurez cliqué sur le bouton**"Créer un format**", le site design reprendra automatiquement la même mise en page template , mais avec les nouvelles dimensions.

Si vous n'êtes pas satisfait du résultat, vous pouvez toujours déplacer les différents composants, redimensionner le logo, ajuster les textes, etc. Lorsque vous êtes satisfait de votre nouveau template, cliquez sur le bouton d'**enregistrement** situé dans le coin supérieur droit.

# **Génération de bannières**

Maintenant que vous avez terminé de créer votre nouvelle marque numérique template, vous pouvez générer des bannières à partir de celle-ci. Pour ce faire, cliquez sur le bouton **Générer des bannières** dans le coin supérieur droit.

Vous serez redirigé vers ce que nous appelons un "entonnoir". Cette page vous permet de configurer vos paramètres et les données utilisées pour le rendu de vos bannières.

# **1. Remplissez le nom de l'opération.**

Il sera utilisé pour créer un nouveau dossier. Chaque bannière générée sera stockée dans ce dossier.

# **2. Sélectionnez les formats que vous souhaitez utiliser.**

Dans ce guide, nous allons sélectionner tous les formats disponibles sur le site template.

# **3. Sélectionnez une méthode de création :**

- **Créer à partir de zéro :** Cette action créera un tableau vide et vous devrez saisir votre contenu manuellement.
- **Importation CSV :** Importez directement votre contenu à partir d'un fichier CSV. Vous pouvez utiliser un document Google sheet qui contient toutes les données nécessaires pour remplir votre template. (Texte, couleurs des images etc..) . Plus d'informations peuvent être trouvées [**ICI**](https://help.abyssale.com/en/article/how-to-import-my-csv-file-into-abyssale-11mhp5k/).
- **Airtable :** Connectez votre base Airtable et importez des données directement depuis celle-ci.


abyssale tâche de génération

# **4. Préparez vos données**

Dans ce guide, nous allons choisir la méthode "créer à partir de zéro" et remplir la colonne que nous voulons mettre à jour.

> Toutes les cellules qui ne sont pas remplies utiliseront la valeur par défaut du site template.

Par exemple, j'ai ajouté 7 lignes pour chaque jour de la semaine, puis j'ai mis à jour le texte, les images et la couleur du bouton pour l'appel à l'action (composant bouton).


Abyssale spreadsheet avec des données de remplacement

5. Lorsque vous êtes prêt à générer des bannières, cliquez sur le bouton " **Enregistrer et fermer**".

6. **Enregistrez et commencez à générer**: Cet entonnoir consommera **28** appels API (7 lignes * 4 formats). Confirmez avant de commencer à générer des bannières.
