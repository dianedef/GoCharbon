# 3 outils gratuits pour trouver les liens cassés sur un site

**Trouver et réparer les liens cassés sur un site est essentiel.** Il est important de maintenir son site Web en bonne santé et de réparer les liens cassés lorsque vous en trouvez. Les liens cassés ou les liens morts peuvent nuire à l’expérience utilisateur de vos visiteurs et à la réputation du site. Les moteurs de recherche comme Google risquent de vous pénaliser et de vous faire payer au prix fort des liens sur votre contenu s’ils ne mènent nulle part. En effet, les moteurs de recherche considèrent généralement les liens cassés comme une indication qu’un site Web est mal entretenu. Il est donc crucial de trouver les liens cassés sur un site et de les traiter rapidement.

Les grands outils SEO proposent quasiment tous dans leurs versions payantes de checker votre site à la recherche de liens morts. Il existe aussi de bons outils gratuits en ligne qui peuvent vous aider à trouver rapidement et à corriger les liens cassés sur votre site Web. En voici trois. **Trois outils gratuits pour vous aider à chercher les liens cassés**.

## Ckeck My Links. Repérer les liens morts sur n’importe quelle page web


**[Check My Links](https://chrome.google.com/webstore/detail/check-my-links/ojkcdipcgfaekbeaelaapakgnjflfglf?hl=fr)** est une extension pour le navigateur Chrome qui va **tester d’un clic n’importe quelle page web affichée dans le navigateur**. Check My Links va parcourir votre page Web et rechercher les liens brisés.  
Lorsque vous éditez une page Web qui comporte de nombreux liens, c’est pratique de pouvoir vérifier rapidement que tous les liens de la page fonctionnent correctement.

Check My Links trouve rapidement tous les liens d’une page Web et les vérifie pour vous. Il met en évidence ceux qui sont valides et ceux qui sont cassés. En bonus, vous pouvez copier tous les mauvais liens dans votre presse-papiers en un seul clic.

## Online Broken Link Checker. Trouver tous les liens cassés dans l’intégralité d’un site web


**[Online Broken Link Checker](https://www.brokenlinkcheck.com/)** est une moulinette simple mais particulièrement efficace pour **ausculter tous les liens présents non pas sur une page mais sur l’intégralité d’un site web.** Que celui-ci compte 10, 100, 1000 ou 10 000 pages. Ce petit utilitaire va parcourir en bon petit soldat toutes les pages de votre site et vous fournir un rapport très détaillé sous la forme d’une sorte de feuille de calcul.  
Vous y trouverez la liste de tous les liens posant un problème sur votre site avec dans l’ordre, le titre du lien, un lien direct pour voir la page où se trouve le lien cassé et le type de message d’erreur obtenu. Simple mais efficace.

## Dr Link Check. Le médecin des liens brisés


**[Dr Link Check](https://www.drlinkcheck.com/)** est l’outil le plus complet de cette liste. Il s’agit d’**un véritable diagnostic archi complet que va vous livrer ce docteur des liens morts une fois que vous lui aurez soumis l’url du site à visiter**. Dr Link Check vous fournit un diagnostic complet. Cela comprend les liens internes aux pages et les liens sortants vers d’autres sites web, ainsi que les liens vers des images, des feuilles de style et d’autres fichiers de ressources.

L’outil fournit un accès facile aux informations dont vous avez besoin pour localiser et réparer les liens sur votre site Web. Les résultats peuvent également être triés et filtrés de différentes manières, et exportés au format CSV pour un traitement ultérieur dans Microsoft Excel. Il est gratuit pour les 1500 premiers liens. Il faudra ensuite passer par la case abonnement pour bénéficier de toute sles options.

En complément, on peut aussi faire appel à des outils tels que [Google Webmaster Tools](https://developers.google.com/search) et Bing Webmaster Tools pour rechercher des liens cassés sur vos sites. Si vous publiez vos contenus sous WordPress, vous pouvez également vous faire aider d’une extension dédiée comme [Broken Link Checker.](https://fr.wordpress.org/plugins/broken-link-checker/) Une fois installée celle-ci va surveiller et tester tous les liens internes et externes de votre site à la recherche de liens brisés.
