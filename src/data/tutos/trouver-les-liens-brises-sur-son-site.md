Les liens brisés ne vont pas ruiner ton SEO à eux seuls. Mais ils dégradent l’expérience utilisateur, cassent ton maillage et donnent l’impression d’un site mal entretenu.

Bonne nouvelle : c’est un problème simple à traiter si tu le fais proprement.

## Ce qu’il faut vérifier

Tu veux repérer :

- les liens internes qui renvoient vers des `404` ;
- les liens externes morts ;
- les images ou ressources cassées ;
- les redirections inutiles en chaîne.

## Trois méthodes utiles

### 1. Crawler ton site

Le plus propre reste un crawler comme :

- Screaming Frog ;
- Sitebulb ;
- Ahrefs Site Audit ;
- ou un outil en ligne si ton site est petit.

C’est la meilleure option si tu veux un diagnostic global.

### 2. Vérifier page par page

Pour un contrôle rapide, une extension de navigateur peut suffire sur une page précise. C’est utile en relecture ou avant publication.

### 3. Regarder Search Console

Search Console ne remonte pas tout, mais elle peut te signaler des URL cassées explorées par Google. Ça aide surtout à prioriser ce qui a un vrai impact.

## Dans quel ordre corriger

Commence par :

1. les liens cassés sur les pages qui génèrent déjà du trafic ;
2. les liens internes vers des pages stratégiques ;
3. les liens présents dans la navigation, le footer ou les blocs récurrents ;
4. les liens externes morts dans tes articles piliers.

## Que faire quand un lien est cassé

Tu as quatre options :

- corriger l’URL ;
- remplacer par une source à jour ;
- supprimer le lien s’il n’apporte plus rien ;
- rediriger proprement si la page a été déplacée.

Le pire choix, c’est de laisser un lien mort “pour plus tard”.

## Les erreurs classiques

- créer une redirection vers une page peu pertinente ;
- empiler les redirections ;
- ignorer les liens dans les anciens articles ;
- faire une grosse migration sans recrawler le site.

## Routine simple

Une fois par mois :

1. crawle le site ;
2. exporte les erreurs ;
3. trie par importance ;
4. corrige les 20 % qui ont le plus d’impact ;
5. recrawle pour valider.

Ce n’est pas la tâche la plus glamour du web. Mais c’est exactement le genre de détail qui rend un site plus propre, plus crédible et plus agréable à utiliser.
