Le code NAF n’est pas parfait, mais il reste très utile pour filtrer une base d’entreprises rapidement.

## À quoi ça sert

- segmenter un marché ;
- cibler une prospection ;
- exclure des secteurs hors sujet ;
- structurer un export.

## Ce qu’il ne faut pas lui demander

- une précision absolue ;
- une compréhension fine de l’offre réelle ;
- une qualification commerciale complète.

## Bonne méthode

1. choisis quelques codes pertinents ;
2. croise-les avec d’autres filtres ;
3. vérifie un échantillon ;
4. affine ensuite.

Le code NAF est un bon premier tamis. Pas un verdict.
