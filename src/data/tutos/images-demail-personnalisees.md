Une image personnalisée peut faire lever un sourcil dans une boîte mail saturée. Mais si elle n’apporte rien ou si elle paraît artificielle, elle devient juste un gadget de plus.

## Quand ça vaut le coup

- prospection ciblée ;
- onboarding ;
- invitation ;
- relance très contextualisée.

## Ce qu’il faut personnaliser

- un prénom ;
- un nom d’entreprise ;
- une donnée de contexte ;
- une mise en situation.

## Ce qu’il faut éviter

- personnaliser pour personnaliser ;
- surcharger l’image ;
- oublier le fallback si les images sont bloquées.

Le bon usage, c’est une personnalisation qui renforce le message. Pas qui se contente de dire “regarde, on peut le faire”.
