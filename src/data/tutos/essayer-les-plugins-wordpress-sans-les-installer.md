Installer un plugin directement sur ton site de prod “pour voir” est une habitude bête et coûteuse.

## Le bon réflexe

Teste dans :

- un site jetable ;
- une préproduction ;
- une copie locale ;
- ou un bac à sable dédié.

## Ce qu’il faut vérifier

- compatibilité avec ton thème ;
- compatibilité avec les autres plugins ;
- impact sur la vitesse ;
- clarté de l’interface ;
- qualité du nettoyage si tu le retires.

## Ce qu’il faut éviter

- tester sur un site client en direct ;
- empiler les plugins “au cas où” ;
- garder un plugin juste parce qu’il fonctionne une fois.

Un plugin doit mériter sa place. Le test sert justement à éviter d’alourdir ton site pour rien.
