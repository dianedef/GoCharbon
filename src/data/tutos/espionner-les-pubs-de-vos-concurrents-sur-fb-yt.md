Tes concurrents dépensent déjà pour tester des messages. Ce serait dommage de ne pas regarder ce qu’ils laissent publiquement derrière eux.

## Outil 1 : Meta Ad Library

La bibliothèque publicitaire de Meta permet de consulter les annonces actives sur Facebook, Instagram et Messenger. Source : [Meta Help Center](https://www.facebook.com/help/259468828226154).

### Comment l’utiliser

1. ouvre la bibliothèque ;
2. choisis le pays ;
3. recherche une marque ou une page ;
4. observe les annonces actives.

### Ce qu’il faut noter

- la promesse principale ;
- le format dominant ;
- les accroches des premières lignes ;
- les CTA ;
- la cohérence entre les variantes.

## Outil 2 : Ads Transparency Center de Google

Pour YouTube et l’écosystème Google, le centre de transparence publicitaire permet de voir les annonces associées à un annonceur. Source : [Google Ads Transparency Center](https://adstransparency.google.com/).

Regarde surtout :

- la répétition des messages ;
- les créas vidéo récurrentes ;
- les accroches textuelles ;
- la durée de présence de certains formats.

## Méthode d’analyse utile

Crée un mini tableau avec :

- concurrent ;
- offre poussée ;
- angle ;
- format ;
- preuve utilisée ;
- CTA ;
- remarque.

Au bout de 15 à 20 pubs analysées, tu verras vite les motifs.

## Ce qu’il faut faire ensuite

Ne copie pas. Reformule.

Cherche :

- un angle plus honnête ;
- une preuve plus forte ;
- une offre plus claire ;
- ou une audience mieux définie.

La veille publicitaire ne sert pas à plagier. Elle sert à éviter de démarrer à zéro.
