- Comment activer/désactiver les plugins ?
    
    Il est possible d'activer et de désactiver les plugins directement depuis le tableau de bord de WP Umbrella.
    
    Pour ce faire, cliquez sur l'onglet Mettre à jour sur le site Web de votre choix.
    
    
    
    Cliquez ensuite sur activer / désactiver.
    
    
    
- Comment fonctionne la mise à jour sécurisée de WP Umbrella ?
    
    Avec WP Umbrella, vous pouvez mettre à jour vos plugins et thèmes en toute sécurité.
    
    Comment cela marche-t-il?
    
    Lorsque vous déclenchez une mise à jour, nous surveillons le processus (erreurs, bugs, succès).
    
    Si la mise à jour échoue ou déclenche des erreurs, nous ramenons automatiquement le plugin à sa version précédente.
    
    Si tout va bien, on continue.
    
    Et c'est tout.
    
    Cela vous fera gagner des tonnes d'heures et rendra la maintenance de WordPress beaucoup moins stressante.
    


Vous pouvez maintenant mettre à jour tous vos plugins sur tous vos sites Web (en un seul clic) et cela devrait vous faire gagner beaucoup de temps ! 🕒

L'" **onglet Mise à jour"** affichera toutes les mises à jour disponibles sur vos sites WordPress.

Les mises à jour disponibles sont regroupées par plugin. Vous pouvez cliquer sur le plugin pour voir le nombre de site(s) concerné(s) par la mise à jour.

Une fois que vous avez sélectionné le plugin/site Web, vous pouvez cliquer sur " **Mettre à jour** " pour déclencher la mise à jour.

Quitter la page n'interrompra pas la mise à jour. Vous recevrez une notification lorsque la mise à jour sera terminée.

- Il semble que nous ne puissions pas communiquer avec l'API WordPress
    
    WP Umbrella utilise l' [API REST de WordPress](https://developer.wordpress.org/rest-api/) pour communiquer avec votre site WordPress. Cela nous permet de connaître, par exemple, les plugins ou le thème qui nécessitent une mise à jour.
    
    Parfois, vous pouvez rencontrer le message :
    
    ```
    Il semble que nous ne puissions pas communiquer avec l'API WordPress.
    
    ```
    
    Cela peut arriver pour deux raisons principales :
    
    1. Votre site n'est pas un site WordPress
    2. Votre hébergeur bloque WP Umbrella, nous ne pouvons donc pas communiquer avec votre API WordPress.
    
    Si votre site Web est propulsé par WordPress, vous devez :
    
    1. Vérifiez si votre plugin de sécurité ne bloque pas l'accès à l'API REST de WordPress.
    2. Sinon, envoyez cet e-mail à votre hébergeur :
    
    > Salut!Nous utilisons un plugin nommé WP Umbrella pour gérer notre thème et nos plugins. Le plugin utilise l'API WordPress REST pour le [faire.Il](http://faire.Il) semble que vous l'empêchez de fonctionner.  Ils reçoivent actuellement 401 erreurs non autorisées.  Pourriez-vous s'il vous plaît mettre en liste blanche ce genre d'URL /wp-jason/wp-umbrella/v1 et leur User-Agent : " parapluie wordpress" ?Merci!Salutations,
    
- Comment mettre à jour les plugins de l'application WP Umbrella
    
    Si vous gérez plusieurs sites WordPress, mettre à jour vos plugins directement dans l'application WP Umbrella vous fera gagner beaucoup de temps.
    
    Dans cet article, je vais vous expliquer comment mettre à jour les plugins en toute sécurité depuis l'application WP Umbrella.
    
    _Nota bene : Nous avons développé une technologie pour sécuriser la mise à jour. Si un problème survient pendant la mise à jour, nous restaurons automatiquement la version précédente. Lisez plus d'informations sur WP Umbrella Safe Update._
    
    # **Comment savoir si vos plugins sont à jour ?**
    
    Accédez à [l'application WP Umbrella](https://app.wp-umbrella.com/) . À partir de là, vous verrez tous les sites Web qui utilisent des plugins obsolètes.
    
    
    
    # **Mettre à jour les plugins en toute sécurité depuis l'application WP Umbrella**
    
    
    
    À partir de là, vous pouvez mettre à jour votre plugin un par un en cliquant sur " Safe Update " ou vous pouvez sélectionner tous vos plugins et les mettre à jour en bloc.
    
    Cliquez ensuite sur appliquer et patientez. Vous recevrez un avis dans le coin supérieur droit lorsque la mise à jour sera terminée.
    
    
    
    C'est ça!
