Créer une facture ne devrait pas te prendre une demi-journée. Le vrai sujet, ce n’est pas la vitesse. C’est d’aller vite **sans oublier les mentions importantes** et sans te bricoler un système bancal.

## Ce qu’il te faut avant de commencer

- tes coordonnées d’entreprise ;
- les coordonnées du client ;
- un numéro de facture cohérent ;
- la date ;
- le détail de la prestation ou du produit ;
- le montant HT, la TVA si tu y es soumis, et le total.

## Outil simple ou modèle propre

Tu peux faire ça de trois façons :

- un logiciel de facturation ;
- un modèle Google Sheets bien tenu ;
- un outil de devis/facture léger si ton volume reste faible.

Le bon choix dépend surtout de ton rythme. Si tu factures souvent, prends un vrai outil. Si tu fais peu de factures, un modèle propre peut suffire.

## Ce qu’il ne faut pas oublier

- une numérotation logique ;
- les bonnes informations légales ;
- un PDF propre ;
- un archivage simple.

## Le bon workflow

1. prépare un modèle unique ;
2. remplis les variables ;
3. exporte en PDF ;
4. classe la facture immédiatement ;
5. note le statut payé / impayé.

Le gain de temps vient surtout de là : un système simple, répété proprement.
