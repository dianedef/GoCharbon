Le web change vite. Si tu dépends d’une information précise, mieux vaut surveiller automatiquement les pages utiles que revérifier manuellement cent fois.

## Ce que tu peux suivre

- prix ;
- stock ;
- texte d’une offre ;
- nouvelles pages ;
- modifications d’une fiche.

## Ce qu’il faut prévoir

- la source ;
- la fréquence ;
- le seuil de bruit acceptable ;
- l’action à prendre.

Une veille saine n’est pas là pour t’alerter sur tout. Elle est là pour te signaler le peu qui compte vraiment.
