Si tu montes déjà tes vidéos dans DaVinci Resolve, tu peux aussi y préparer une miniature propre. Pas besoin d’ouvrir trois logiciels pour faire une image correcte.

## Commence par le bon frame

Choisis une image :

- nette ;
- lisible ;
- expressive ;
- bien éclairée.

Un frame moyen ne devient pas une bonne miniature par miracle.

## Ensuite, ajoute seulement l’essentiel

- un titre très court ;
- une hiérarchie visuelle claire ;
- un contraste fort ;
- éventuellement un élément graphique récurrent.

## Ce qu’il faut éviter

- trop de texte ;
- trop d’effets ;
- une miniature qui ne correspond pas au contenu ;
- un visage illisible ou flou.

## Dans DaVinci

Tu peux :

1. extraire un still ;
2. retravailler l’image ;
3. ajouter du texte ;
4. exporter au bon format.

Le plus important n’est pas l’outil. C’est la clarté de la miniature dans un petit rectangle vu sur mobile.
