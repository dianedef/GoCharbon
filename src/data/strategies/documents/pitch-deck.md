# Pitch deck : structurer une présentation qui aide vraiment à convaincre

Le pitch deck n'est pas un concours de slides. C'est un support de conversation. Il doit t'aider à raconter un projet de manière claire, ordonnée et crédible.

## Les blocs qui reviennent presque toujours

- le problème ;
- la solution ;
- le marché ;
- le produit ;
- le modèle économique ;
- la traction ;
- la concurrence ;
- l'équipe ;
- la demande ou l'étape suivante.

## Adapte le deck au lecteur

Un investisseur, un partenaire, un client ou un banquier ne regardent pas le même deck avec les mêmes attentes. Le squelette peut rester proche, mais l'accent doit changer.

## Les erreurs à éviter

- slides surchargées ;
- storytelling héroïque sans preuve ;
- trop de détails techniques ;
- marché gonflé ;
- concurrence inexistante par magie.

Un bon deck n'essaie pas d'avoir réponse à tout. Il doit surtout rendre le projet lisible et ouvrir la discussion au bon niveau.
