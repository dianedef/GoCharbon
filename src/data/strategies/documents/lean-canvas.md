# Lean canvas : modéliser une idée sans s'enfermer dans un business plan prématuré

Le lean canvas est utile quand une idée existe, mais que tout n'est pas encore stabilisé. Il oblige à poser en une page les éléments qui comptent vraiment : problème, cible, proposition de valeur, modèle économique et canal.

## Pourquoi il reste utile

Parce qu'il force à clarifier vite :

- pour qui tu construis ;
- quel problème tu traites ;
- ce que tu proposes ;
- comment tu peux gagner de l'argent ;
- ce qui rend l'idée fragile ou crédible.

## Les blocs qui méritent le plus d'attention

- problème ;
- segments de clients ;
- proposition de valeur ;
- canaux ;
- revenus ;
- coûts ;
- métriques ;
- avantage défendable.

## Ce qu'il faut éviter

- écrire des généralités interchangeables ;
- remplir les blocs sans lien entre eux ;
- prendre le canvas pour une validation ;
- passer deux jours dessus au lieu d'aller confronter l'idée au terrain.

## Ce que le lean canvas fait bien

Il ne prouve pas qu'une idée marchera. En revanche, il révèle très vite si l'idée est encore trop floue pour être testée correctement. C'est déjà très précieux.
