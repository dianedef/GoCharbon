### 2. Block IP address used by CDN Networks

CDN, Content Delivery Networks is a system of distributed servers (network) that deliver web pages and other Web content to a user based on the geographic locations of the user. In the context of YouTube, when you watch a video, the video will be played from the server that the video was previously cached and stored, but not from the YouTube’s server. YouTube buffering speed might vary when different videos are played from different cache servers. You can block the IP addresses used by these CDN networks so that the video starts buffering from the original YouTube server itself which is comparatively faster than CDN servers.

### Block CDN IP Addresses on Windows

- Click on Start button and search for CMD. If you’re a Windows 8 or 8.1 user, then press Windows key and type “CMD”.
- Simply, right click on CMD icon and hit Run as Administrator.
- Just copy the below-given code in your CMD and press enter.

netsh add firewall firewall add rule name=”YouTubeTweak” dir=in action=block remoteip=173.194.55.0/24,206.111.0.0/16 enable=yes

- Now, you can close the Command prompt windows and restart your browser and the internet connection.

### 3. Use YouTube DNS

You can increase your YouTube buffering speed by using internet connection through different DNS. Using Youtube open DNS will help you in speed up the youtube video buffering.

For Windows XP users:

- Go to Control Panel from the Start menu.
- Click on Network Connections from the Control Panel choices.
- Choose your connection from the Network Connections window.
- Click on Properties button.
- Select Internet Protocol (TCP/IP) and click Properties.
- Now, click the radio button. Use the following DNS server addresses and type in OpenDNS addresses in the Preferred DNS server and Alternate DNS server fields.
- 208.67.222.222
- 208.67.220.220

For Windows 7/8/8.1 Users:

- Go to Control Panel. Click on Network and Internet then click on Network and Sharing Center.
- Click on Change adapter settings.
- Now, Right Click on Local Area Connection then select Properties button.
- Double click on Internet Protocol version 4 (TCP/IPv4) (don’t change anything in IP address).
- Click the radio button. Use the following DNS server addresses and type
- Preferred DNS server: 208.67.222.222
- Alternate DNS server: 208.67.220.220
- Click Ok and you’re done.

### 6. Increase the Cache Size for Flash Player

This trick might not work for Google Chrome users as the browser compels YouTube to use HTML5 for playing videos. However, the trick works well on Internet Explorer and Mozilla Firefox. You can load videos faster on YouTube by increasing the cache size for the [**YouTube videos**](https://www.alltechbuzz.net/download-youtube-videos-for-offline-browsing/) on your PC or Laptop which is nothing but increasing the virtual RAM for the videos. This can load videos faster as the flash player utilize more space from your device storage.

Steps to Increase the size of cache for flash player

- Open Mozilla or Explorer browser and go to YouTube.
- Start watching any video and pause it after a while.
- Now, Right click on the video and click on Settings.

  

- A small menu pops up on the video screen. Just click on the small yellow folder icon at the bottom of the box.
- Now take the slider all the way to right i.e. “maximum” and close the pop-up box.
- Lastly, restart your browser and internet connection. You can observe some improvement in your YouTube buffering speed.
