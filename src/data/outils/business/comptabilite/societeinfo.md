# SocieteInfo

## INTELLIGENCE COMMERCIALE B2B : LA BASE DE DONNÉES DE 12 MILLIONS D'ENTREPRISES FRANÇAISES

### tl;dr

SocieteInfo est une plateforme française de prospection et d'intelligence commerciale B2B, créée en 2016. Elle centralise les données de 12 millions d'entreprises françaises en combinant Open Data et technologies Big Data. Recherche d'entreprises, enrichissement de fichiers, extraction d'emails, API entreprise. Utilisée par 100 000+ professionnels. Considérée comme la meilleure solution de données B2B en France dans 90% des cas.

**Note** : outil de prospection B2B, pas un logiciel comptable. Mal catégorisé ici.

### 📢 Alternative à

Dropcontact, Pappers, Société.com, Kompass.

## ✔️ Bénéfices

- **12M+ d'entreprises** : la base la plus complète de France
- **Enrichissement de fichiers** : CA, effectif, emails, téléphones
- **Mises à jour quotidiennes** : données toujours fraîches
- **API puissante** : intègre les données dans tes outils

### 🎯 Pour Qui ?

- Commerciaux et responsables prospection
- Équipes marketing B2B
- Entrepreneurs cherchant des prospects qualifiés
- Développeurs intégrant des données entreprises

## 🛠️ Fonctionnalités

- **Recherche avancée** : filtres par secteur, localisation, CA, effectif, forme juridique
- **Enrichissement Excel** : upload ton fichier, récupère CA, effectif, emails, dirigeants
- **Emails de contacts** : extraction des adresses professionnelles
- **API Entreprise** : accès programmatique aux données
- **Fiches entreprises** : infos légales, financières, dirigeants, actionnaires, comptes annuels
- **Alertes** : suivi d'entreprises, notifications de changements

API REST complète. Export CSV/Excel. Compatible avec tous les CRM et outils de prospection.

## 🎖️ L'équipe et l'Histoire

Créée en 2016, SocieteInfo a profité de l'ouverture des données publiques (Open Data) pour construire une base de données B2B exhaustive. La plateforme combine Big Data, web scraping et données officielles (INSEE, Infogreffe, BODACC) pour fournir les informations les plus à jour sur les entreprises françaises.
