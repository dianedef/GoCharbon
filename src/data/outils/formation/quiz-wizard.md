# Quiz Wizard

## Génération de quiz à partir de contenus existants

Quiz Wizard permet de transformer rapidement des notes, cours ou documents en QCM et questions. C’est efficace pour valider la compréhension sans écrire chaque question à la main.

## Quand c’est utile
- Préparer un contrôle de compréhension après un module.
- Créer une banque de questions cohérente pour une classe ou une équipe.
- Générer plusieurs versions pour éviter la mémorisation mécanique.

## Méthode recommandée
1. Charge un document propre (objectifs + concepts clés).
2. Génère un premier lot de questions.
3. Filtre les questions trop faciles / floues / ambiguës.
4. Sauvegarde une version courte et une version longue.

## Limites
- La qualité de sortie dépend de la qualité des sources fournies.
- L’IA aide la rédaction, pas la validation pédagogique complète.
- Révise systématiquement les distracteurs et formulations.
