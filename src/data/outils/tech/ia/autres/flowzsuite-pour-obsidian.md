## TubeFlowz

### 1. Gestion Intelligente des Vidéos YouTube
- Import facile via URL YouTube (plusieurs formats d'URL supportés)
- Support des vidéos individuelles et des playlists
- Extraction automatique des métadonnées
- Organisation avec système de tags personnalisés
- Bibliothèque de vidéos avec recherche rapide

### 2. Modes d'Affichage Flexibles
- Mode onglet (tab)
- Mode barre latérale (sidebar) 
- Mode superposition (overlay)
- Interface responsive avec hauteur ajustable
- Thème adapté à Obsidian

### 3. Contrôles de Lecture Avancés
- Lecture/Pause
- Contrôle du volume et mode muet
- Mode plein écran
- Vitesse de lecture personnalisable (0.25x à 16x)
- Vitesse favorite configurable
- Navigation par timestamps
- Mode de lecture configurable (streaming/téléchargement)

### 4. Intégration Obsidian Native
- Bouton dans la barre latérale avec menu déroulant
- Raccourcis clavier personnalisables
- Insertion de timestamps cliquables dans les notes
- Création de liens YouTube avec timestamp
- Support complet du format Markdown

### 5. Fonctionnalités IA (avec OpenAI)
- Génération automatique de résumés
- Création de notes structurées
- Extraction des points clés
- Traduction automatique du contenu

### 6. Configuration et Personnalisation
- Choix du mode d'affichage par défaut
- Configuration du mode de lecture préféré
- Réglage de la vitesse de lecture favorite
- Gestion de l'affichage des recommandations YouTube
- Personnalisation des templates de notes
- Gestion sécurisée des API keys

### 7. Support Multilingue
- Interface en Français et Anglais
- Détection automatique de la langue
- Messages d'erreur localisés

### 8. Architecture Technique Robuste
- Architecture modulaire avec services séparés
- Gestion d'état centralisée
- Support des tests unitaires
- Système de cache pour les métadonnées
- Gestion robuste des erreurs
- Protection des API keys
- Validation des URLs YouTube

### 9. Performance et Optimisation
- Chargement différé des composants
- Mise en cache des résultats API
- Interface utilisateur optimisée
- Traitement asynchrone des tâches lourdes

### Bénéfices Principaux
1. **Productivité Améliorée**
   - Prise de notes facilitée pendant le visionnage
   - Organisation automatique du contenu
   - Gain de temps grâce aux fonctionnalités IA

2. **Expérience Utilisateur Optimale**
   - Interface intuitive et flexible
   - Personnalisation poussée
   - Intégration transparente avec Obsidian

3. **Fiabilité et Sécurité**
   - Gestion robuste des erreurs
   - Protection des données sensibles
   - Architecture stable et testée

4. **Accessibilité**
   - Support multilingue

- Détection automatique du type de flux

### 📂 Organisation
- Système de groupes pour classer les flux
- Structure de dossiers automatique dans Obsidian
- Deux modes de sauvegarde des articles :
  - Mode fichier unique (tous les articles d'un flux dans un seul fichier)
  - Mode multi-fichiers (un fichier par article)
- Tags et catégories automatiques

### 📖 Mode Lecture
- Interface dédiée à la lecture
- Navigation fluide entre les articles (⌘←/→)
- Marquage des articles lus/non lus
- Système de filtrage par groupe/feed
