# What is the cost of translation ?

Translation is a new feature that we added on August 2022.

The cost of translation is an additional cost to the transcription.

For a 60 minutes long file, the transcription will cost 60 minutes of credits and if the user wants to translate the file, it will use another 60 minutes. In total it will cost 120 minutes in credits for a full translated file.

The cost of the translation are per language. If the user decide to translate 60 minutes file in 3 language it will cost 60 credits (for transcription) and 60*3 = 180 credits (for translation). A total of 240 credits.

# What format can I download my audio from Noota to my computer ?

Today Noota use OGG format to create high quality small size file. So when you choose to download a file from Noota it will be downloaded in OGG.

If you want to download the video, you can download the video from the subtitle mode as MP4 file.

How long does it take for my transcription to be done ?

**Did you know:** Manual transcription usually take 4 hours for 1 hour of recording

For an uploaded file **under 2 hours**, it will take approximately 1 hour (or less) of processing (half of the original duration).

If your file is **more than 2 hours long**, then it can take up to the same duration as the file itself !
