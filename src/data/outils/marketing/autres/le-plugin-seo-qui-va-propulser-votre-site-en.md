la vérité sur les plugins de SEO

Les « plugins SEO » tuent-ils votre site Web ?

Mais, Brendan, j'ai « fait le référencement » sur mon site Web.

Je ne comprends pas pourquoi ça ne marche pas…

Il y a environ trois mois, j'ai fait un petit commentaire dans le PS de ma newsletter :

« De plus, dans tant de ces fils de discussion, les gens mentionnent les plugins (comme Yoast) de manière interchangeable avec le référencement.

Je m'en voudrais de ne pas mentionner que ces plugins (malgré de grandes promesses) ne font rien d'important pour le référencement autre que de vous aider à définir une méta description.

Pour de nombreuses personnes, « faire le référencement » signifie rendre les lumières vertes sur leur plugin de référencement.

Quelques personnes ont répondu à cet e-mail qu'elles étaient surprises car ces plugins se présentent comme « SEO » et font également des déclarations générales pour vous aider à améliorer votre classement.

J'ai même analysé pourquoi la page d'accueil d'un de ces plugins (Rank Math) était trompeuse (au mieux) ou une arnaque (au pire).
