# Cibles sémantiques

Les cibles sémantiques sont la clé qui donne aux LLM comme GPT-4 vision la capacité d'exécuter des actions précises. Atlas-1 a été formé pour détecter les éléments de l'interface utilisateur susceptibles de correspondre à l'intention de la cible suivante, afin d'activer les "cibles sémantiques", qui distillent les cibles jusqu'à leur intention sous-jacente dans le langage plutôt que dans le code. [Auparavant, les outils d'automatisation utilisaient des stratégies de ciblage telles que les sélecteurs CSS, Xpath et même la vision par ordinateur, mais toutes ces stratégies échouaient lorsque les services mettaient à jour leur conception.Puisque les cibles sémantiques traduisent toujours les cibles potentielles de la même façon, même si vous changez complètement la conception du site Web, cela permet enfin de construire des automatisations robustes et à l'épreuve du temps. Puisque ces cibles sémantiques opèrent dans le langage, cela permet également aux LLM comme GPT-4 de les utiliser directement pour effectuer des actions précises.
