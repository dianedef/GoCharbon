lf (list files) comparé à Ranger et Joshuto :

- Configuration minimale
- Scriptable facilement

- Performances rapides
- Portable
- Peu de dépendances

Inconvénients :
- Interface basique
- Moins de fonctionnalités prêtes à l'emploi

Recommandé pour :
- Utilisateurs avancés
- Personnes qui aiment personnaliser
- Ceux qui veulent un gestionnaire de fichiers minimaliste

En résumé : excellent outil mais demande plus d'investissement personnel que Ranger.

J'ai déjà mentionné fm dans une réponse précédente, mais voici un résumé :

**fm par mistakenelf** :

Avantages :
- Écrit en Go (performances correctes)
- Interface TUI simple
- Léger et portable
- Code propre

Inconvénients :
- Projet encore jeune
- Fonctionnalités limitées
- Moins de personnalisation
- Communauté restreinte
- Documentation basique

**Positionnement** :
- Plus simple que xplr ou lf
- Moins mature que Ranger
- Bon pour débuter avec les gestionnaires TUI

**Verdict** : Correct pour un usage basique, mais pas encore assez mature pour remplacer les solutions établies.
