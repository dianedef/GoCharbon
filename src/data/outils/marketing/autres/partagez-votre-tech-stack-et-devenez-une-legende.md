The tech stack file une nouvelle façon d'organiser, de visualiser et de rendre utiles les données de ta pile technologique dans toutes les applications. Avec cela, nous avons trois nouvelles annonces de produits, alors accrochez-vous !

À la base, le Tech Stack File est un fichier YML unique (techstack.yml) détaillant la pile technologique complète et la chaîne d'approvisionnement logicielle d'un repo Git, avec des métadonnées sur chaque technologie. Il est complet, couvrant l'open source, les logiciels commerciaux, le SaaS et l'infrastructure. C'est comme une nomenclature logicielle améliorée (SBOM), mais plus complète et lisible par l'homme.

Dans la plupart des entreprises, les données relatives à la pile technologique sont dispersées dans des feuilles de calcul et des wikis. En collaboration avec la Cloud Native Computing Foundation (CNCF), notre objectif avec The Tech Stack File est d'établir une norme universelle pour les données relatives à la pile technologique. Ce n'est que la première version, nous aurons besoin de tes commentaires pour en faire quelque chose d'utile pour tout le monde. Ce fichier est disponible dès maintenant - tu peux obtenir tes propres fichiers de piles technologiques pour n'importe lequel de tes dépôts GitHub grâce à la toute nouvelle application GitHub GRATUITE Stack File. L'application GitHub génère automatiquement un techstack.yml pour chacun de tes dépôts et le visualise également dans un magnifique fichier Markdown appelé techstack.md.


L'application GitHub maintient ces fichiers à jour au fur et à mesure que tes piles technologiques évoluent grâce à des Pull Requests automatisées.
 Tu peux considérer cela comme des documents automatisés sur les piles technologiques ! Le fait d'avoir des données sur la pile technologique facilement accessibles dans GitHub facilite l'intégration de la base de code, les migrations à grande échelle et les tâches de diligence juridique/conformité.

StackShare AI & StackShare Connect

Pour tirer pleinement parti de ce nouveau format de fichier, nous annonçons StackShare AI - un moyen d'exploiter l'ensemble de la base de connaissances de StackShare avec une personnalisation intégrée par le biais de tes nouveaux fichiers de pile technologique. Imagine que tu puisses demander "qui dans mon entreprise est passé à React v18.2.0 au cours des deux derniers mois ?" - via un bot Slack conversationnel. StackShare AI sera disponible pour les entreprises début 2024, mais tu peux [t'inscrire sur la liste d'attente dès aujourd'hui](NITUHWOIBVCGPYD).

Enfin, nous annonçons StackShare Connect - un moyen facile pour toi d'infuser les données de ta pile technologique dans les apps et les sites Web de tous les jours. Nous sommes fiers de nous associer à FOSSA, la plateforme de gestion Open Source native pour les développeurs, en tant que premier fournisseur de StackShare Connect. Bientôt, tu pourras prendre tes fichiers Tech Stack et les intégrer à ton compte FOSSA pour superposer des politiques de conformité et de gouvernance.
