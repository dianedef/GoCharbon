# WP Search Console

## GOOGLE SEARCH CONSOLE DANS WORDPRESS : LE PLUGIN FRANÇAIS QUI AFFICHE TES DONNÉES SEO SANS QUITTER TON DASHBOARD

### tl;dr

WP Search Console, c'est le **plugin WordPress français** qui **affiche les données de Google Search Console directement dans ton dashboard WordPress**. Plus besoin d'aller sur Google Search Console séparément — tu vois tes **impressions, clics, CTR, position moyenne, requêtes, pages** directement dans WordPress. Développé en **France**. Le concept : tu connectes ton compte Google Search Console via l'API, et le plugin affiche les données dans un dashboard intégré à WordPress. Tu peux filtrer par période, par page, par requête. C'est un pont entre Google Search Console et WordPress — tu restes dans ton environnement de travail habituel. Cible : **blogueurs, webmasters WordPress, consultants SEO** qui gèrent des sites WordPress et veulent consulter les données SEO sans quitter l'admin.

### Alternative à

Google Search Console (interface web), MonsterInsights (volet GSC), Rank Math (volet GSC intégré), Site Kit by Google. WP Search Console se distingue par son **focus exclusif Google Search Console** (pas un plugin multi-fonctions qui dilue les données), son **interface simple** dans WordPress (pas besoin de naviguer dans l'interface complexe de Google Search Console), et sa **légèreté** (un plugin qui fait une chose bien, pas une usine à gaz).

## Bénéfices

**Données dans WordPress** — Tu n'as plus besoin d'aller sur Google Search Console. Les données (clics, impressions, CTR, position) sont affichées directement dans ton dashboard WordPress. C'est un gain de temps au quotidien.

**Interface simplifiée** — Google Search Console est puissant mais complexe. Le plugin présente les données essentielles dans une interface claire et simple. Les KPIs qui comptent, sans le bruit.

**Filtres** — Filtre par période (7j, 28j, 3 mois, 6 mois), par page, par requête. Tu identifies rapidement les pages qui performent et celles qui déclinent.

**Léger** — Un seul plugin, une seule fonction. Pas d'impact sur les performances de ton site WordPress.

### Pour Qui ?

- Blogueurs WordPress qui veulent suivre leur SEO facilement
- Webmasters qui gèrent plusieurs sites WordPress
- Consultants SEO qui veulent un aperçu rapide des données GSC
- Rédacteurs qui veulent voir les requêtes qui amènent du trafic sur leurs articles
- PME qui ont un site WordPress et veulent comprendre leur SEO

## Comment Utiliser WP Search Console

1. **Installation** — Installe le plugin via WordPress > Extensions > Ajouter
2. **Connexion** — Connecte ton compte Google (OAuth) pour accéder aux données GSC
3. **Dashboard** — Consulte les données dans ton menu WordPress
4. **Analyse** — Filtre par page, requête, période
5. **Optimisation** — Identifie les pages à optimiser et les requêtes à cibler

## Fonctionnalités

### Dashboard

- **Clics** — Nombre de clics depuis Google
- **Impressions** — Nombre d'affichages dans les résultats
- **CTR** — Taux de clic moyen
- **Position** — Position moyenne dans les SERPs
- **Graphiques** — Évolution dans le temps

### Pages

- **Top pages** — Pages qui génèrent le plus de clics
- **Pages en déclin** — Pages qui perdent des positions
- **Opportunités** — Pages en position 5-20 (potentiel d'optimisation)
- **Filtrage** — Par URL, par période

### Requêtes

- **Top requêtes** — Mots-clés qui amènent du trafic
- **Nouvelles requêtes** — Requêtes récemment apparues
- **CTR par requête** — Identification des requêtes à faible CTR (title/meta à optimiser)
- **Position par requête** — Suivi de la position pour chaque mot-clé

### Administration

- **Multi-sites** — Compatible WordPress multisite
- **Cache** — Mise en cache des données pour les performances
- **Export** — Export des données

## Prix

- **Gratuit** — Version de base avec les fonctionnalités essentielles
- **Premium** — Fonctionnalités avancées (filtres, exports, données historiques)

## L'équipe et l'Histoire

- Plugin WordPress développé par un développeur connu seulement sous le prénom "François". Aucune entité juridique (SIREN) identifiable
- Le plugin s'inscrit dans l'écosystème des plugins SEO WordPress (Yoast, Rank Math, Site Kit) mais se concentre uniquement sur l'affichage des données GSC
- **⚠️ Statut incertain** — En mars 2026, le site wpsearchconsole.com renvoyait une erreur 521 (serveur Cloudflare injoignable). Le produit pourrait être **abandonné**. À ne pas confondre avec le plugin "Search Console" de Tropicalista (Francesco Pepe, italien) sur WordPress.org

## Cas d'Usage Concrets

**Le blogueur WordPress** — Tu écris 4 articles par semaine. Chaque matin, tu ouvres ton dashboard WordPress. WP Search Console te montre les articles qui ont le plus de clics cette semaine, les requêtes qui montent, et les articles en déclin. Tu sais exactement quels articles mettre à jour.

**L'agence web (10 sites clients WordPress)** — Tu gères 10 sites WordPress. Au lieu d'ouvrir 10 onglets Google Search Console, tu consultes les données directement dans chaque admin WordPress. 2 minutes par site au lieu de 5.

## Points Forts

- Données Google Search Console dans WordPress
- Interface simple et claire
- Plugin léger (pas d'impact sur les performances)
- Gratuit en version de base
- Développé en France
- Filtres par page, requête, période

## Points Faibles

- **Données limitées à GSC** — Pas d'analyse de backlinks, pas de keyword research, pas d'audit technique. C'est un afficheur de données GSC, pas un outil SEO complet
- **Dépendant de l'API GSC** — Si Google change son API, le plugin peut casser
- **WordPress uniquement** — Si tu n'es pas sur WordPress, ce plugin ne te sert à rien
- **Site Kit by Google** — Google propose son propre plugin officiel (Site Kit) qui affiche les données GSC + GA + AdSense. La concurrence officielle est rude
- **Mises à jour** — Les plugins WordPress doivent être régulièrement mis à jour. Un plugin abandonné peut devenir une faille de sécurité
