    - 1/ Use the keyword "related:" to find websites similar to the main website. If you want to search for websites similar to a specific website just Type - related:domain
    - 2/ Use the Keyword "search:" to search within a website. Looking for a page from a specific website? Just "search:" and your query.
    - 3/ Use the keyword "*" to replace missing words. The asterisk symbol is used for wild card searches in Google. "iPhone and * alternatives" - will search for the same quoted text but it will replace the * with anything it can fine.
    - 4/ Use double quotes to search for exact matching word results. For example: "netsuite alternative" will force Google to find pages that use these exact phrase.
    - 5/ Use "@" to find Twitter, Instagram, and other social accounts.
    - 6/ Use three dots "..." to search within a range. It can be years or price range. TV $800...$1000 - will show TV in this range. Mailbox 1992...1998 - will show you pages during these years.
    - 7/ Find search results using synonyms of a word using the "~" operator. Want to find searches who use Tutorials, Guides, or courses on automation. Type "Automation ~tutorials"
    - 8/ Write "time <city>" to see the time in a city.
    - 9/ Use "define: < keyword >" to get the definition of any word. I will write another thread on how I automate these basic searches using Alfred on Mac.
    - 10/ Search based on filetype. If you want to search for docs, ppts, CSV, or excel files on a specific search, use this.

## tu peux pas te rater si tu suis la checklist seo
- 
- Assure-toi d'inclure le mot-clé dans la première phrase
- Essaie de placer chaque phrase sur une nouvelle ligne :
	- Cela facilite la lecture = Taux de rebond plus bas = Meilleur classement + plus de revenus.
- Ajoute des images : Utilise [https://unsplash.com/](__MASK_5__) - n'utilise pas d'images protégées par des droits d'auteur. C'est pourquoi tu devrais utiliser Unsplash.
- Si tu le peux, incorpore quelques vidéos pertinentes de YouTube.
- Formate joliment le texte :
	- Ajoute des paragraphes, des interlignes, insère des liens, inclus les vidéos YouTube.
		- Les vidéos/images sont optimales tous les 200 à 300 mots
		- Ajoute des méta-descriptions avec ton mot-clé : o
			- Par exemple, ton titre/titre SEO
			- Un excellent plugin pour cela est Yoast : Il est gratuit à utiliser sur WordPress et il t'aide pp.!lmjsg ton contenu.
		- L'introduction doit être courte et aller droit au but
		- Ajoute des liens internes/externes pertinents : o
			- Vise 2 à 3 liens internes/externes 0
- Assure-toi de cibler des mots-clés à faible concurrence :
	- Et chaque page ne doit cibler qu'un seul mot-clé
- Crée un aperçu de ce que tu dois écrire :
	- Cherche sur Google le mot-clé pour lequel tu essaies de te classer, par exemple "meilleurs produits pour chats" > Regarde les titres que tes concurrents ont utilisés > Utilise également ces titres dans ton article.
	- Recherche un mot-clé sur Google > Récupère les FAQ de Google dans la section " Les gens demandent aussi " > Colle-les au bas de ton article et réponds-y.
	- Un outil utile pour cela est NeuronWriter (lien aff)
- Rédige ensuite le contenu de chaque rubrique :
	- Essaie de viser un peu plus de mots que tes concurrents sur Google pour ce mot-clé, sans fioritures ni remplissage.
- Assure-toi que ton article correspond à l'intention de recherche de Google :
	- Par exemple, si l'article le mieux classé sur Google est un article de liste pour ton mot-clé, tu ferais aussi un article de liste.
	- Si l'article le mieux classé est une page d'avis, tu ferais aussi des avis sur des produits.
- Mentionne le mot-clé pour lequel tu essaies de te classer, dans ton contenu, plusieurs fois :
	- Cela doit sembler "naturel" et non forcé
