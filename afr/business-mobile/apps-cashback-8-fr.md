## 1. Macadam

**Type :** Cashback sur achats en ligne

### Comment ça marche ?

1. Connecte ton compte Macadam
2. Fais tes achats via l'app
3. Macadam détecte l'achat
4. Tu reçois le cashback sur ton compte
5. Tu retires quand tu atteins le seuil

### Cashback par catégorie

| Catégorie    | Cashback |
| ------------ | -------- |
| Voyages      | 10-15%   |
| Électronique | 5-8%     |
| Mode         | 8-12%    |
| Restaurants  | 5-10%    |
| Autres       | 3-6%     |

### Avantages

✅ Cashback jusqu'à 15%
✅ Beaucoup de partenaires
✅ Interface simple
✅ Notifications push pour les offres
✅ Programme de parrainage

### Inconvénients

❌ Seuil de retrait : 5€ minimum
❌ Délai de retrait : 1-2 mois
❌ Certaines catégories ont des % bas
❌ Parfois l'app plante

### Pour qui ?

- Ceux qui font beaucoup d'achats en ligne
- Voyageurs réguliers
- Célibataires qui commandent souvent

### Verdict

**Note : 7.5/10**

Macadam est fiable et a un bon cashback. Le seuil de 5€ est raisonnable. Les délais de retrait sont corrects.

## Stratégie d'Optimisation

### 1. Utilise plusieurs apps

Ne te limite pas à une app.

**Combinaison recommandée :**

- Macadam ou Armado (cashback standard)
- FreeCash (cashback + micro-tâches)
- BeMyEye (enquêtes)

### 2. Active les notifications

Toutes les apps notifient les offres spéciales.

**Pourquoi ?** Les offres temporaires peuvent aller jusqu'à 20-30% de cashback.

### 3. Vérifie le cashback AVANT chaque achat

Ouvre l'app avant d'acheter.

**Pourquoi ?** Tu peux voir si l'offre est dispo et optimiser ton cashback.

### 4. Cumule les offres

Certaines apps ont des bonus de parrainage, des bonus inscription, etc.

**Stratégie :** Enregistre-toi sur plusieurs apps, cumule les bonus, puis garde seulement les plus rentables.

### 5. Retire automatiquement

Ne garde pas l'argent sur les apps.

**Pourquoi ?** Tu peux oublier, l'app peut fermer, ou tu peux être tenté de dépenser.
